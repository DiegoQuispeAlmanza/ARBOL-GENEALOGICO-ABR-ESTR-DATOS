El presente proyecto forma parte del desafio ABR del curso de Estructura de Datos

===== Generación de Arboles binarios de búsqueda para generar árboles genalógicos =====

Nuestra propuesta del ABR grupal, consta en la conceptualizacion, diseño, creacion, prototipado y documentación de un "Sistema de Gestión y
Visualización de árboles generacionales para representar relaciones genéticas entre familias de gatos mediante árboles binarios de búsqueda"

Con el objetivo de poder visualizar y representar sobre como la genetica es un factor que influye y determina los fenotipos dentro de una
familia de gatos, y sobre el como estos permiter relacionar, estudiar e identificar ancestros o parintes comunes.
La investigacion genetica permite poder identificar factores medicos, mutaciones y evoluciones de los genes dentro de las familias; pudiendo
predecir la herencia fisica, tendencias psicologicas e incluso identificar patrones.
